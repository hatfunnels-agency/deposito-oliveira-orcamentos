# Depósito Oliveira - Sistema de Orçamentos

Sistema web de orçamentos para Depósito Oliveira, com integração Bling ERP e persistência via Supabase.

## Funcionalidades
- Catálogo de produtos integrado com Bling API v3 (OAuth 2.0)
- Cálculo de frete via API ViaCEP
- Geração de orçamentos com código único ORD-XXXXXXX
- Persistência de clientes e orçamentos no Supabase
- Histórico de orçamentos com busca e filtros
- Gestão de status de pedidos

## Configuração

Veja `.env.example` para as variáveis de ambiente necessárias.

Deploy automático via Vercel + GitHub. Env vars configuradas.
